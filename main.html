Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is all about value of HTML content, including tags and innerText is about value of visible text, ignoring hidden elements.Finally, at last text content is all about managing text, including hidden elements, without formatting.

Question-3: What is event delegation in the DOM?

Event delegation uses a parent node to handle node child events efficiently via event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and moves up to its ancestors in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?
